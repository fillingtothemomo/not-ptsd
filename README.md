# not-ptsd
## Clone the repo
```
git clone git@github.com:fillingtothemomo/not-ptsd.git
```

## Download the dependencies
```
pip install -r requirements.txt
```
run the backend server 
```
python manage.py runserver
```
run the frontend server

```
npm run dev

```
